## 追加ルール

- ファイル名は `YYYYMMDD_<テーマ>.md` 形式とする。
- 同じテーマで複数回分析する場合は連番を付ける（例: `20260503_trouble_analysis_2.md`）。
- 分析結果の対応が完了した場合でも削除せず、`対応済み` のラベルをファイル内に記載する。
