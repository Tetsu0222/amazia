# `_archive/phaseX-11/` について

本フォルダは phaseX-11（ドキュメント駆動の多層化）の **棚卸し作業ログ** を保管する場所。
AI が能動的に参照することを想定しない（`docs/ai_context/` の参照ハブ群からも導線を張らない）。

## 配置の意図

phaseX-11 Step 1 で実施したドキュメント棚卸しは、Step 2 以降の「CLAUDE.md 参照ハブ化」「Hook 設計」「skill 化」「データ化」の **設計判断の根拠** として残す価値がある。
ただし、棚卸しログ自体を AI が会話毎に読みに行く必要はない（参照ハブ・規約は別ファイルが担う）。

そのため：

- `docs/ai_context/` 配下には置かない（AI が能動 Read する候補に紛れ込ませない）
- `docs/_archive/` 配下に隔離し、人間が振り返りで参照する位置付けにする
- 連番フェーズ（phaseX-11 / phaseX-12 ...）ごとにサブフォルダで分離

## 内容

- [_doc_inventory.md](_doc_inventory.md) — 7 ファイル分の棚卸しテーブル + 重複・課題リスト

## 取り扱いルール

- 削除しない（次フェーズの設計判断の根拠として参照可能にしておく）
- 内容更新が必要な場合は、棚卸し時点の値を残しつつ「YYYY-MM-DD 追記」形式で末尾追記する（上書きしない）
