# フェーズ7：一括削除・一括編集

## ステータス
✅ 完了（2026-05-03）

## 範囲
- Amazia Console

## 機能概要

### 一括削除
- 商品一覧にチェックボックスを追加
- 1件以上チェックした状態で「一括削除」ボタンが活性化
- 削除確認ダイアログを表示してから実行

### 一括編集（在庫数）
- チェックボックスで複数商品を選択
- 「一括編集」ボタンを押すとモーダルを表示
- モーダル内にも商品ごとのチェックボックスを用意
- 在庫数をまとめて変更できる（個数のみ対象）

## 画面構成

```
商品一覧画面
├── [全選択チェックボックス]
├── [一括削除ボタン] [一括編集ボタン]  ← チェック時に活性化
└── 商品行ごとにチェックボックス

一括編集モーダル
├── 対象商品リスト（チェックボックス付き）
│   ├── □ 商品A  在庫: [___]
│   ├── □ 商品B  在庫: [___]
│   └── ...
└── [キャンセル] [保存]
```

## TDDテストケース（JUnit / Amazia Core）
- 複数商品IDを指定して一括削除できること（DELETE /api/products?ids=1,2,3 → 204）
- 一括在庫更新が反映されること（PATCH /api/products/bulk-stock → 200）

## TDDテストケース（PHPUnit / Amazia Console）
- チェックなし状態では一括削除ボタンが非活性であること
- 一括削除後に一覧が更新されること
