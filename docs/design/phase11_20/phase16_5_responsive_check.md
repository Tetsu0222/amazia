## E2E ゴールデンパス（vitest e2e テストで自動化）

`src/test/cart_flow.e2e.test.jsx` で以下のシナリオを自動実行する：

- [ ] 商品一覧表示 → 商品詳細遷移 → カート追加 → カートページ → Checkout `?from=cart` → 注文完了画面
- [ ] 検索バーで `?q=` クエリ → 結果一覧 → 0 件時に「すべての商品を見る」リンクで一覧復帰
